Lab Report Cover Page Generator

A web based application to generate a clean, professional, single page A4 lab report cover.
The project is designed for university students who want a properly formatted academic cover page without using Word or Canva.

The generated cover page always fits 100 percent inside one A4 page and is ready for printing or PDF export.

Features

Single page A4 output only

No extra print pages

Responsive layout for mobile, tablet, and desktop

Editable course, student, and teacher information

Horizontal Submitted by and Submitted to sections with exact spacing

Minimal academic design without boxes

Print and PDF ready layout

Dark mode editor for comfortable editing

Works fully in the browser

Technologies Used

HTML

CSS

JavaScript

No external libraries or frameworks are required.
How to Use

Open the website link

Fill in course name, course code, student name, roll, teacher name, designation, and submission date

Preview the cover page instantly

Click Download as PDF

In print settings

Select A4 paper size

Set margins to None

Turn off headers and footers

Save as PDF or print
